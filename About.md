##  About This Project

This project is a full-stack web application designed to simplify customer account management, billing, and transaction tracking for small businesses.

>  My primary role in this project was **deployment, infrastructure setup, and system management**, rather than developing the application from scratch.

### Deployment & Infrastructure Work

* Deployed an existing **React frontend and Node.js backend** application
* Managed and configured a **RESTful API service** in a production-like environment
* Connected and maintained **MongoDB Atlas** cloud database

### DevOps & Deployment Work

* Provisioned and managed **AWS EC2 (IaaS)** instance
* Containerized services using **Docker & Docker Compose**
* Configured **Nginx as a reverse proxy** for routing frontend and backend
* Implemented **HTTPS using Let's Encrypt SSL certificates**
* Set up **custom domain and subdomain routing** (frontend + API separation)
* Monitored and troubleshooted deployment issues (networking, ports, DNS)

### Future Scalability (AWS Enhancements)

To handle higher traffic and improve availability, the following AWS services can be integrated:

* **Elastic Load Balancer (ELB)** → Distribute traffic across multiple instances
* **Auto Scaling Group (ASG)** → Automatically scale EC2 instances based on load
* **Amazon CloudFront** → CDN for faster global content delivery
* **Amazon RDS / DynamoDB** → Managed database for better scalability and reliability
* **Amazon S3** → Store static assets (images, frontend build)
* **AWS CloudWatch** → Monitoring, logging, and alerts
* **AWS Route 53** → Advanced DNS management and health checks
